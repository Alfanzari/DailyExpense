## Deskripsi

DailyExpense adalah aplikasi pencatat pengeluaran harian yang dibangun menggunakan Kotlin dan Jetpack Compose. Pengguna dapat mencatat, mengelola, dan memantau pengeluaran sehari-hari dengan tampilan yang bersih dan intuitif.

## Fitur

- Tambah, lihat, edit, dan hapus data pengeluaran
- Tampilan list dan grid yang bisa diubah sesuai preferensi
- Konfirmasi sebelum menghapus data
- Recycle Bin — data yang dihapus bisa dipulihkan kembali
- Preferensi tampilan tersimpan menggunakan DataStore

## Teknologi

- Kotlin & Jetpack Compose
- Room Persistence Library
- Navigation Compose
- DataStore Preferences
- MVVM Architecture
